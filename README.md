# scripts

add `export PATH=$PATH:~/scripts/` to your ~/.bashrc

usage:

1.stress test:

  run command `stress test expected generator [number_of_tests]`

2.generate contest:

  runcommand `cf_gen contest_id [problem_letter]`
  
 for example:
  `cf_gen 1557`
  `cf_gen 1557 ABC`

3.run sample:

  run command `runcase [problem letter]`

for example:
  `runcase A`
